# E-Commerce Sales Dashboard

## Project Overview
This project showcases the creation of a dynamic sales dashboard for e-commerce data using Python and Power BI. The dashboard provides actionable insights into sales trends, profits, returns, and performance across regions and categories.

## Features
- Cleaned and preprocessed 500+ e-commerce orders using Python and Pandas.
- Added calculated metrics, including Profit Margin.
- Power BI measures for:
  - Total Sales
  - Total Profit
  - Total Returns
  - Overall Profit Margin %
- Interactive dashboard with:
  - KPI Cards
  - Region-wise and Category-wise charts
  - Daily sales trend line chart
  - Returns breakdown pie chart
  - Slicers for Region, Category, Returned, and Date
